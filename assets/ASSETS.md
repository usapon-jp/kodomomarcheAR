# Asset Naming

Use these category-based file names under `assets/`.

- Face accessories: `assets/face/face_<name>_<nn>.png`
- Earring pairs: `assets/face/face_earring_<name>_<nn>.png`
- Characters: `assets/character/character_<name>_<nn>.png`
- Frames: `assets/frame/frame_<name>_<nn>.png`
- Backgrounds: `assets/background/background_<name>_<nn>.png`
- Stamps: `assets/stamp/stamp_<name>_<nn>.png`
